# 🐱 Random Cats

A simple web application that displays random cat images using the CATAAS API.

## Features

- Display random cat images
- Generate a new random cat
- Responsive design
- Uses HTML, CSS, and JavaScript
- Uses Async/Await and Fetch API

## API

https://cataas.com/cat

## Technologies

- HTML
- CSS
- JavaScript
